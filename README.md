## Задача 1

### Текст задачи

На основе данного списка строк получить новый, где к каждой строке в конце дописан заданный символ.

### Алгоритм решения

Была описана функция `InputUserString` для создания списка, заданного пользователем, с предварительной проверкой значения `size` на корректность размера ввода. **Пустые строки в списке допускаются**. Далее считывается **один** символ с клавиатуры с проверкой на пустую строку. В List.map совершается конкатенация строк из списка и заданного пользователям символа.

# Задание 2

## Задача 1

### Текст задачи

Список содержит строки. Найти самую короткую строку

### Алгоритм решения

Была описана функция `InputUserString` для создания списка, заданного пользователем, с предварительной проверкой значения `size` на корректность размера ввода. **Пустые строки в списки допускаются**. Далее была описана функция `shortest_string` с использованием свертка списка List.fold. В качестве начального значения была взята голова списка, а в качестве свертка весь остальной список (хвост). В результате в `acc` оказывается кратчайшая строка.
